# Timetable

## Расписание занятий
```
Реализованные страницы:
 Ученик, преподаватель, деканат:
  -Главная
  -Расписание
  -Смайлик открывает расписание факультета ИВТ
 Преподаватель:
  -Вход в аккаунт
  -Выход из аккаунта
  -Личный кабинет(выводится личное расписание)
 Деканат:
  -Вход в аккаунт
  -Выход из аккаунта
  -Изменение/удаление выбранного пункта расписания
  -Добавление пункта расписания для дня недели определённой группы
  -Личный кабинет: добавление/удаление/изменение группы
 
Не реализованные страницы:
 Деканат:
 -Личный кабинет: добавление/удаление преподавателя, администратора
   
Что ещё сделать:
 -Хранить всю информацию в СУБД или продумать сохранение информации
 -Изменить/улучшить оформление, классы, методы
 -Провести тестирование
```
