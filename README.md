# Timetable

## Расписание занятий
```
Реализованные страницы:
 Ученик, преподаватель, деканат:
  -Главная
  -Расписание
  -Смайлик открывает расписание факультета ИВТ
 Преподаватель:
  -Вход в аккаунт
 Деканат:
  -Вход в аккаунт
  -Изменение выбранного пункта расписания
 
Не реализованные страницы:
 Преподаватель:
 -Личный кабинет(можно выводить личное расписание)
 Деканат:
 -Личный кабинет(куча кнопок различной функциональности)
  -Добавление/удаление преподавателя, администратора, группы, расписания
   (Для расписания проверить, что не существует пара с тем же номером)
   (Удаление расписания для каждого пункта и с выбором)
```
