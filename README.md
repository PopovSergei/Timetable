# Timetable

## Расписание занятий
```
Реализованные страницы:
 Ученик, преподаватель, деканат:
  -Главная
  -Расписание
  -Смайлик открывает расписание факультета ИВТ
 Преподаватель:
  -Вход в аккаунт
 Деканат:
  -Вход в аккаунт
  -Изменение выбранного пункта расписания
  -Добавление/удаление пункта расписания для дня недели определённой группы
 
Не реализованные страницы:
 Преподаватель:
 -Личный кабинет(можно выводить личное расписание)
 Деканат:
 -Личный кабинет(куча кнопок различной функциональности)
  -Добавление/удаление преподавателя, администратора, группы
   
Что ещё сделать:
 -Хранить всю информацию в СУБД или продумать сохранение информации
 -Изменить/улучшить оформление, классы, методы
 -Провести тестирование
```
