# Timetable

## Расписание занятий
```
Реализованные страницы:
 Ученик, преподаватель, деканат:
  -Главная
  -Расписание
 Преподаватель:
  -Вход в аккаунт
  -Выход из аккаунта
  -Личный кабинет(выводится личное расписание)
 Деканат:
  -Вход в аккаунт
  -Выход из аккаунта
  -Расписание:
    -Изменение/удаление выбранного пункта расписания
    -Добавление пункта расписания для дня недели определённой группы
  -Группы: добавление/удаление/изменение группы
  -Пользователи: добавление/удаление/изменение пользователя
  -Личный кабинет: ссылки на расписание, группы, пользователей
  
Пароль от аккаунтов-примеров: 123
  
Ещё реализовано:
 -База данных
```
